üéØ Introducing the WHERE Clause

The WHERE clause is like a filter that helps you find specific rows in your table based on conditions.

üìù Basic WHERE Syntax

SELECT column1, column2 FROM table_name WHERE condition;

üîç How WHERE Works

Step What Happens Example
1 SQL looks at each row Checking row by row...
2 Tests the condition Is price > 500?
3 If TRUE, includes the row Yes! Include this row
4 If FALSE, skips the row No, skip this row

‚ö° Comparison Operators

= (Equal)

price = 100

Finds exact matches

!= (Not Equal)

Excludes matches

> (Greater Than)

price > 500

Values above threshold

< (Less Than)

stock < 10

Values below threshold

>= (Greater or Equal)

rating >= 4.5

Values above or equal

<= (Less or Equal)

year <= 2000

Values below or equal

üèãÔ∏è Practice Exercises

Exercise 1: Basic Filtering

Task: Find all products in the 'Phone' category

-- Write your query here: SELECT * FROM products WHERE _________;
SELECT * FROM products WHERE category = 'Phone';

Exercise 2: Price Filtering

Task: Find all products priced above $500

-- Write your query here: SELECT name, price FROM products WHERE _________;
SELECT name, price FROM products WHERE price > 500;

Exercise 3: Stock Check

Task: Find products with low stock (less than 30 items)

-- Write your query here: SELECT name, stock FROM products WHERE _________;
SELECT name, stock FROM products WHERE stock < 30;

Exercise 4: High-Rated Products

Task: Find products with rating 4.5 or higher

-- Write your query here: SELECT name, rating FROM products WHERE _________;
SELECT name, rating FROM products WHERE rating >= 4.5;

Exercise 5: Exclude Category

Task: Find all products that are NOT laptops

-- Write your query here: SELECT name, category FROM products WHERE _________;
SELECT name, category FROM products WHERE category != 'Laptop';
